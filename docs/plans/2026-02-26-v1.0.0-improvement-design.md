## 解决方案

### 1. Dashboard 统计 API

**新增后端 API**：

```
GET /api/v1/site/dashboard/stats
```

**响应**：
```json
{
  "success": true,
  "data": {
    "posts": {
      "total": 150,
      "published": 120,
      "draft": 20,
      "scheduled": 10
    },
    "users": {
      "total": 25,
      "active": 23,
      "inactive": 2
    },
    "comments": {
      "total": 300,
      "pending": 15,
      "approved": 280,
      "spam": 5
    },
    "media": {
      "total": 500,
      "storage_used": "2.5GB"
    }
  }
}
```

**数据库查询**：
- posts: `SELECT status, COUNT(*) FROM site_xxx.sfc_site_posts GROUP BY status`
- users: 从 `sfc_users` + site关联查询
- comments: 从 `sfc_site_comments` 查询
- media: 从 `sfc_site_media_files` 查询 storage

### 2. PostEditor 分类/标签选择器

**方案**：Popover + Combobox（shadcn/ui）

**组件结构**：
```
PostEditor
├── TitleInput
├── ContentEditor (BlockNote)
├── CategorySelect (Popover + Combobox)  ← 新增
├── TagSelect (Popover + Combobox)       ← 新增
├── SeoSettings
└── PublishSettings
```

**组件设计**：
- `CategorySelect`: Popover 弹出树形分类选择，支持多选
- `TagSelect`: Popover 弹出 Combobox，支持搜索、新建标签

### 3. PostsTable 状态切换

**方案**：在每行添加操作下拉菜单

**新增操作**：
- 发布 (Publish)
- 取消发布 (Unpublish)
- 定时发布 (Schedule)
- 移至回收站 (Trash)

**UI 设计**：
- 每行末尾添加 `...` 按钮
- 点击展开操作菜单

### 4. Settings 错误处理

**方案**：显示详细错误信息

**改进点**：
- 字段级别错误提示
- API 返回的具体错误信息展示
- 保存前验证
